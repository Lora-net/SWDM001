Check the parameters, and in case of success, generate the digest summary which contains important size info.

Parameters
[in]paramssx126x LR-FHSS parameter structure
[in]hop_sequence_idSpecifies which hop sequence to use
[in]payload_lengthLength of application-layer payload
[out]statesx126x LR-FHSS state structure that will be initialized by this function
Remarks
It is not necessary to explicitly call this function if the helper function sx126x_lr_fhss_build_frame is used. If the preprocessor symbol HOP_AT_CENTER_FREQ is defined, hopping will be performed with PA ramp up/down, but without actually changing frequencies.
Returns
