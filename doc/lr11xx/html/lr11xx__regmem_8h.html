Write words into register memory space of LR11XX.

A word is 32-bit long. The writing operations write contiguously in register memory, starting at the address provided.

Parameters
[in]contextChip implementation context
[in]addressThe register memory address to start writing operation
[in]dataThe buffer of words to write into memory. Its size must be enough to contain length words.
[in]lengthNumber of words to write into memory
